# create-vite-w-cli

快速创建vite + vue3的项目脚手架, 内部集成axios, pinia, element-plus, tailwind, eslint, sass, typescript

## 使用
```shell
npm create vite-w-cli@latest
```
or

```shell
npm create vite-w-cli@latest --project-name my-project
```
